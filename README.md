## Ejecutar en desarrollo

1. clone el repositorio
2. ejecute `npm install`
3. tener instalado nest cli

```
npm i -g @nestjs/cli
```

4. levantar la base de datos con docker

```
docker-compose up -d
```

5. ejecute `npm run start:dev`

## Stack usado

- NestJS
- MongoDB
